Regular Expressions in Python

Starter Code

import re
pattern = re.compile(r'raw string')
result = pattern.search("String to search")
result.group()

Alternative method

import re
pattern=r'raw string'
result=re.search(pattern,"String to search")

Grouping with Parentheses

import re
pattern = r'(\d\d\d)-(\d\d\d-\d\d\d\d)'
result = re.search(pattern,"My number is 415-555-4242")
result.group(1)
>>>'415'
result.group(2)
>>>'555-4242'
result.group(0)
>>>'415-555-4242'
result.group()
>>>'415-555-4242'
result.groups() //groups() returns a tuple
>>>('415','555-4242')

Character Class

Shorthand character class Represents
\d Any numeric digit from 0 to 9
\D Any character that is not a numeric digit from 0 to 9
\w Any letter, numeric digit or the underscore character "_"
\W Complement of \w
\s Any space,tab, or newline character
\S Complement of \s

Special Charaters

Special Character Description Example Usage
. Wildcard. matches any character except newline
include "re.DOTALL" in search(pattern,string,re.DOTALL) to make . match any character including newline(\n)

pattern = r'.at'
result = re.findall(pattern,'rat,cat,eat')
>>>['rat','cat','eat']

//Greedy Search: Python is greedy by default
and will try to match the longest string possible
pattern = r'\(.*\)'
result = re.search(pattern,'(test)(2)')
result.group()
>>>'(test)(2)'

//Non-Greddy Search: pattern = r'\(.*?\)'
result = re.search(pattern,'(test)(2)')
result.group()
>>>'(test)'

^ In '[^]' used to indicate the complement of the set
In normal pattern, used to indicate that a match must occur in the beginning of a text

//Normal case
pattern = r'^Hello' result = re.search(pattern,'Hello, I am Wayne')
result.group()
>>>'Hello'


//Character class complement
pattern = r'[^aeiouAEIOU]'
result = re.findall(pattern,'abcde')
>>>['b','c','d']

$ Similar to ^ but match at the end of the text
* Match zero or more of the group that preceds this star

pattern = r'Bat(wo)*man'
result = re.search(pattern,'The adventures of Batwowowoman')
result.group()
>>>'Batwowowoman'

+ Match one or more. It is not optional unlike *

pattern = r'Bat(wo)+man'
result = re.search(pattern,'Batman')
result.group()
>>>None

? Optional Matching. Match zero or one of the group preceding this question mark
Note: 0 or 1

pattern = r'Bat(wo)?man'
result = re.search(pattern,'The adventures of Batman')
result.group()
>>>'Batman'

{} Match repeated groups of specific number of times. (Ha){3,5} means the pattern of (Ha) 3 to 5 times
"{3,}"" means 3 or more
"{,3}" means 3 or less

//(Ha){3,5} == (((Ha)(Ha)(Ha))|((Ha)(Ha)(Ha)(Ha))|((Ha)(Ha)(Ha)(Ha)(Ha)))
pattern = r'(Ha){3}'
result = re.search('HaHaHa')
result.group()
>>>'HaHaHa'

result1 = re.search(pattern,'Ha')
result1.group()
>>>None

//Greedy Search: Python is greedy by default
and will try to match the longest string possible
//Non-Greddy Search: pattern = r'(Ha){3,5}?'
result = re.search(pattern,'HaHaHaHaHa')
result.group()
>>>'HaHaHa'

[] Used to define your own set of characters

pattern = r'[aeiouAEIOU]'
vowel = re.findall(pattern,'abcde')
>>>['a','e']


pattern = r'[a-zA-Z0-9./?_$^,*.]'
result = re.findall(pattern,'aA0./?_$^,*.')
>>>['a', 'A', '0', '.', '/', '?', '_', '$', '^', ',', '*', '.']

\ Used for escape character and character class

pattern = r'\d'
result = re.findall(pattern,'a1b2c3')
>>>['1','2','3']

| This is called Pipe. It is like a or operator. Starting from right, it will return the first match of the pattern

pattern = r'Batman|Tina Fey'
result = re.search(pattern,'Batman and Tina Fey')
result.group()
>>>'Batman'

pattern = r'Batman|Tina Fey'
result1 = re.search(pattern,'Tina Fey and Batman')
result1.group()
>>>'Tina Fey'

pattern = r'Bat(man|mobile|copter|woman)'
result = re.search(pattern,'Batmobile lost a wheel')
result.group()
>>>'Batmobile' result.group(1)
>>>'mobile'

() Group characters

Note

To match reserved characters like "(", use "\" as an escape character. e.g "\("

The findall() Method

Return all matches as a list

//Without group
pattern = r'12'
result = re.findall(pattern,'12312')
print(result)
>>>['12','12']

//With group
pattern = r'(1)(2)'
result = re.findall(pattern,'12312')
print(result)
>>>[('1','2'),('1','2')]

Case Insensitive Matching

Include "re.I" or "re.IGNORECASE" as an argument into re.search and re.findall to ignore case sensitivity

The sub() Method

Usage:
pattern = r'Agent \w+'
replace = 'CENSORED'
result = re.sub(pattern,replace,'Agent Alice killed Agent Bob')
>>>'CENSORED killed CENSORED'

Using Matched Groups

pattern = r'(\w)\d'
replace = r'\1*'
result = re.sub(pattern,replace,'a1b2c3')
print(result)
>>>'a*b*c*'

Managing Complex Regexes

You can spread the regular expression over multiple lines
by including "re.VERBOSE" in the re.compile(pattern,re.VERBOSE)

Include more than 1 argument in compile

Use bitwise operator "|"
re.compile(pattern,re.IGNORECASE|re.DOTALL|re.VERBOSE)
